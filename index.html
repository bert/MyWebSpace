Hi, welcome here.
Here is a modest start with generating developer information for the gEDA-suite.
The information is generated with Doxygen version 1.4.6.
Another project related to gEDA is pcb-gfpw, a GTK based footprint generator for pcb.
The name is derived from GTK footprintwizard.
There is also a Command Line Interface (CLI) version named fpw derived from footprintwizard (current status: work in progress).
With this application one can create footprint files used by the pcb layout editor.
Any comments, additions, changes, improvements are welcome.
Then there are some pcb plug-ins in the downloads directory.
  • lockelements.c : to lock/unlock all elements in a pcb layout.
  • findelement.c : to find an element on the current pcb you are editing.
  • upth2pth.c : to change unplated holes to plated holes (Status: work in progress, so YMMV :-)
  • ghellopcb.c : to create a GTK dialog window showing a "Hello World" message (a proof of concept attempt)
